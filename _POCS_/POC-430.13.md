## §4. 🔬 KEY TECHNICAL INSIGHT: Presigned URLs

**Source:** Surge ⚡ — Score 95/100

### The Traffic Flow That Changes Everything

Formbricks v5 uses **S3 Presigned URLs** for file uploads. The client browser uploads DIRECTLY to the S3 bucket. The Formbricks Node.js server never touches the file payload.

**What this means for DigitalOcean:**

| Concern | Reality |
|:--------|:--------|
| DO egress fees for file storage? | **$0.00** — files bypass the droplet entirely |
| Droplet OOM risk from uploads? | **$0.00** — 2GB droplet only handles API calls, not multipart uploads |
| DO vendor lock-in for storage? | **$0.00** — migrate Formbricks off DO → point at same S3 bucket. Zero data gravity. |

### The Architecture

```
[USER BROWSER] ──(1) API request──> [FORMBRICKS DO DROPLET]
                                      │
                                      ├──(2) Generates Presigned URL
                                      │
[USER BROWSER] <──(3) Returns URL────┘
      │
      ├──(4) DIRECT UPLOAD (bypasses DO) ──> [B2 / R2]
      │
      └──(5) DIRECT DOWNLOAD (bypasses DO) <── [B2 / R2]
```

### Why This Is Decisive

> **"The key to this decision isn't just where the storage lives; it's how the traffic flows."** — Surge ⚡

Because the traffic flows DIRECTLY between user browser and storage provider, the DO droplet is irrelevant for file storage. The only question is: which S3-compatible provider gives us the best cost, hybrid architecture, and ecosystem alignment?

## §9. 🏆 RECOMMENDATION: 3-Phase Strategy

### Phase 1 (W30-W34) — Backblaze B2 🥇

**Consensus: 5 of 8 MetaCouncil members recommend B2 for Phase 1.**

| Element | Detail |
|:--------|:--------|
| **Storage** | Backblaze B2 via Cloudflare Bandwidth Alliance |
| **Cost** | ~$0.30/mo at 50GB — effectively free for Phase 1 |
| **Setup** | **15 minutes** — create B2 bucket, generate App Key, set 4 Formbricks env vars |
| **Egress** | $0 — Cloudflare Bandwidth Alliance makes B2→CF→user egress FREE |
| **Exit Option** | S3 API standard — switch to MinIO or R2 with credential swap, zero code changes |
| **Data Flow** | `User → Cloudflare (proxied) → B2 (BW Alliance = free egress)` |

### Phase 2 (W35+, 200GB+) — Migrate to MinIO 🥈

**Consensus: 6 of 8 MetaCouncil members recommend MinIO as Phase 2/3.**

| Element | Detail |
|:--------|:--------|
| **Trigger** | Monthly storage exceeds ~200GB, or sovereignty needs justify ops overhead |
| **Migration** | `rclone sync` from B2 → MinIO. Update 4 Formbricks env vars. Done. |
| **Cost** | ~$6/mo for separate DO droplet (1GB) or Hetzner VPS (€4/mo) |
| **Benefit** | Self-managed encryption keys, full data sovereignty, zero vendor lock-in |

### Phase 3 (W40+, Decentralized) — Storj 🥉

**Consensus: 3 of 8 MetaCouncil members recommend Storj as long-term vision.**

| Element | Detail |
|:--------|:--------|
| **Trigger** | Cooperative ethos demands true decentralization. Storj S3 gateway matures. |
| **Migration** | `rclone sync` from MinIO → Storj. Update 4 Formbricks env vars. |
| **Cost** | ~$0.004/GB/mo + $0.007/GB egress |
| **Benefit** | **True decentralized storage.** No single vendor. Cryptographic access control. |
